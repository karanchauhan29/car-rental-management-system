# 🚗 Car Rental System

A full-stack MERN application for online car booking.

## ✨ Features
- User Authentication
- Book Cars
- View Bookings

## 🛠 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## ⚙️ Setup

### Backend
cd backend  
npm install  
npm start  

### Frontend
cd frontend  
npm install  
npm start
